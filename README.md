## 教具運行方式

利用RFID教具卡，讀取並匯入一連串指令，其中可刪除、新增、儲存等等，執行即可讓機械手臂執行一連串相對應之指令，此外指令對應代碼由0-36, A-G等字母所組成，L,M,R以及1,2,3亦有相對應之涵義。
